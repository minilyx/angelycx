Junior Frontend Developer
Full Time Hybrid — UK Studio £22,000 – £26,000
Now Hiring
+
The Role

You'll be building and maintaining client-facing websites and web applications as part of our frontend team. Working closely with senior developers and designers, you'll write clean HTML, CSS and JavaScript and contribute to React-based projects.

This is a hands-on role from day one. You'll attend client briefings, participate in sprint planning, and have your code reviewed by experienced developers who want to help you grow.

Day-to-Day Responsibilities
  • Building responsive, accessible web interfaces from design mockups
  • Writing semantic HTML5, modern CSS3 and vanilla JavaScript
  • Working with React components alongside senior developers
  • Using Git for version control and collaborating via pull requests
  • Debugging cross-browser issues and ensuring performance
  • Participating in daily stand-ups and sprint retrospectives
  • Writing basic unit tests and maintaining documentation
Required Qualifications
Education (one of)
Level 3 IT / Web Development Diploma, BTec in Computing, A-Level Computer Science, or equivalent
Alternative Routes
Coding bootcamp certificate, freeCodeCamp / Odin Project completion, or demonstrable self-taught portfolio
Core Skills Required
HTML, CSS, JavaScript fundamentals, basic Git usage, understanding of responsive design
Nice to Have
React basics, Figma/design tools, experience with any CSS framework, basic command line
What We're Really Looking For
  • Curiosity — you read documentation for fun and Google things obsessively
  • A portfolio — even small personal projects show us how you think
  • Communication skills — you can explain your thinking clearly
  • Growth mindset — you see feedback as a gift, not a criticism

Trainee Full-Stack Developer
Full Time Hybrid — UK Studio £20,000 – £24,000
Now Hiring
+
The Role

Our trainee full-stack position is designed for people who are brand new to the industry but have demonstrated aptitude across both frontend and backend technologies. You'll rotate across different parts of our stack, getting exposure to everything from database queries to CSS animations.

You'll be paired with a senior mentor throughout and we expect this role to transition to a Mid Developer position within 18–24 months based on performance.

Day-to-Day Responsibilities
  • Contributing to both frontend UI work and backend API development
  • Writing and querying SQL and NoSQL databases under supervision
  • Assisting in building RESTful APIs using Node.js or Python
  • Maintaining and debugging existing client applications
  • Writing and running basic test suites
  • Documenting your code and contributing to technical wikis
  • Shadowing senior developers in client calls and technical planning
Required Qualifications
Education (one of)
HND/HNC in Computing, BSc Computer Science (2:2 or above), Level 4 Web Development qualification, or bootcamp certificate
Alternative Routes
Strong self-taught portfolio covering both frontend and backend work. GitHub showing real projects considered
Core Skills Required
HTML/CSS/JS, basic knowledge of at least one backend language (Node, Python, PHP), basic SQL
Nice to Have
Any framework experience (Express, Flask, React), exposure to REST APIs, familiarity with Linux command line
What We're Really Looking For
  • Breadth of interest — you enjoy both front and back, or are actively learning both sides
  • Problem solving — you don't give up when something breaks, you dig in
  • A real portfolio — show us something you built, even if it's small
  • Coachability — you absorb feedback and apply it quickly

Junior UI / CSS Developer
Full Time Hybrid — UK Studio £21,000 – £25,000
Now Hiring
+
The Role

If you obsess over pixel-perfect layouts, love CSS animations, and care deeply about how things look AND how accessible they are — this role is for you. You'll work closely with our design team to translate Figma mockups into beautiful, performant HTML/CSS.

Day-to-Day Responsibilities
  • Translating design mockups into responsive HTML/CSS layouts
  • Building and maintaining a shared component library
  • Implementing CSS animations and micro-interactions
  • Running accessibility audits (WCAG compliance)
  • Cross-browser testing and visual QA
  • Collaborating with designers in Figma
Required Qualifications
Education (one of)
Any web/IT qualification at Level 3+, Graphic Design qualification with coding component, or equivalent experience
Core Skills Required
Advanced HTML5, CSS3 (including Flexbox, Grid, animations), basic JavaScript, understanding of accessibility
Nice to Have
Figma, Tailwind CSS, SASS/SCSS, any knowledge of WCAG standards, basic React or Vue
Portfolio Essential
We must see examples of your CSS work. Codepen, GitHub, live sites — anything you've actually built
